# Job Discovery Freshness

Use for Oanh's twice-weekly Monday/Thursday job scan.

## Constraints

- Return only 3-5 opportunities.
- Mix actual applications and networking/outreach leads.
- Avoid depressing volume. Ruthless filtering is a feature.
- Prefer fresh/company-confirmed roles over stale LinkedIn listings.
- Do not recommend downlevel Google roles unless Oanh explicitly asks.

## Target lanes

Strong:
- AI learning / EdTech product.
- AI health / regulated public good.
- Creative AI / interactive learning.
- Founder, creator, family, or student-facing AI products.
- Internal AI tools with real product ownership.

Bridge:
- Agentic workflow / AI transformation if implementation-heavy and senior.
- Google internal L6+ AI roles if team/comp are strong.

Avoid:
- Generic strategy & ops.
- Chief of staff / product ops.
- GTM-only / sales-support AI roles.
- Generic AI adoption workshops/change-management consulting.
- Junior/downlevel roles.

## Search sources

Prefer:
- Company career pages.
- Government/public-sector AI orgs: GovTech, MOE, MOHT, HTX, etc.
- Universities / learning orgs.
- AI startups and VC portfolio job boards.
- Recruiter/hiring manager posts.
- LinkedIn as a lead source only; verify elsewhere when possible.

## Freshness checks

For each role, try to confirm:
- Posting is live on company site.
- Posted/reposted date if visible.
- Role location/remote/hybrid.
- Seniority not obviously too junior.
- Application path exists.

## Output format

For each opportunity:
- **Role / org / link**
- **Type:** apply / outreach / watch
- **Why this is in the batch**
- **Risks / caveats**
- **Suggested next step**

End with:
- **My top 1-2 picks**
- **What I skipped and why**
