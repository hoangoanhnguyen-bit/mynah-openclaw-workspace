# Client Follow-Up Rescue System — Pitch Brief for Jeweler

## 30-second opener

“I know the jewelry work itself is not the problem. The painful part is tracking all the promises, deadlines, client messages, repairs, custom orders, and follow-ups across WhatsApp and memory. What I’d like to offer is a lightweight AI assistant setup that helps make sure fewer things slip — not by replacing you, but by acting like a very persistent admin/project coordinator.”

## The problem I heard

From what you shared, the biggest pain points are:

- Missed messages
- Missed deadlines
- Angry or anxious clients
- Too many open projects to track mentally
- Follow-ups living across WhatsApp, notes, memory, and scattered systems
- No easy daily view of “what needs attention today”

The risk is not just admin stress. It affects trust. In jewelry, clients are emotionally invested — engagement rings, gifts, repairs, heirlooms, custom pieces. When they don’t hear back, anxiety becomes anger quickly.

## What I’m proposing

A **Client Follow-Up Rescue System**: a small custom assistant/workflow that helps capture client commitments, track deadlines, surface overdue items, and draft follow-up replies.

It can start very simply, with no big software migration.

The goal:

> Fewer missed promises. Faster replies. Calmer clients. Less mental load.

## What the S$1,200 minimum includes

This is not just “installing a bot.” The price covers setup, customization, testing, and handover.

### 1. Workflow diagnosis

I’ll spend time understanding how projects actually move through your business:

- Where client messages come from: WhatsApp, Instagram, email, walk-ins
- Types of jobs: custom pieces, repairs, resizing, sourcing, cleaning, redesigns
- Typical stages: inquiry → quote → deposit → design → production → fitting → pickup/delivery
- Common failure points: missed replies, missed supplier updates, unclear deadlines, client chasing

Output: a simple map of your current workflow and the points where reminders/AI support can help.

### 2. Project tracking template

I’ll create a lightweight tracker for active client jobs. It can include:

- Client name
- Contact channel
- Project type
- Current status
- Promised next step
- Due date
- Owner/person responsible
- Last client contact
- Next follow-up date
- Risk flag: overdue / waiting on client / waiting on supplier / urgent

This can live in Google Sheets, Notion, Airtable, or another simple tool depending on what you already use.

### 3. AI assistant setup

I’ll configure an assistant that can help with tasks like:

- Turn rough notes into structured project entries
- Extract follow-up items from pasted messages
- Draft polite WhatsApp replies to clients
- Summarize client/project status
- Generate a daily “what needs attention today” list
- Flag overdue or high-risk items
- Create reminder text for follow-ups

Example:

You paste:

> “Mrs Tan asked if the ring resizing will be ready before Friday. Supplier says maybe Thursday. Need to update her tomorrow.”

Assistant turns it into:

- Client: Mrs Tan
- Project: Ring resizing
- Deadline: Friday
- Next action: Update client tomorrow
- Risk: Supplier uncertainty
- Draft reply: “Hi Mrs Tan, just a quick update — we’re checking with the workshop and currently expecting it around Thursday. I’ll confirm again with you tomorrow so you’re not left waiting.”

### 4. Reminder and follow-up system

We’ll set up a practical cadence:

- Daily list of client follow-ups
- Overdue item flags
- Upcoming deadline reminders
- Optional end-of-day “what is still open?” check

The point is to make the system annoyingly helpful — the assistant should catch things before clients do.

### 5. Trial with real active projects

We’ll test it on a small batch first:

- 5–10 active client jobs
- 2-week pilot
- Adjust based on what actually helps
- Remove anything that feels too cumbersome

The system should fit the way you work, not force you into a heavy enterprise tool.

### 6. Handover and training

I’ll show you how to use it in plain language:

- How to enter/update a client job
- How to ask the assistant for today’s priorities
- How to generate reply drafts
- How to review overdue items
- How to keep the tracker clean

Includes a simple written guide.

## What this is not

This is not:

- A full custom app
- A CRM migration
- A replacement for staff judgment
- A fully automated WhatsApp bot that replies to clients without approval
- A magical system that works without anyone updating it

It is a practical operational support layer.

The first version is deliberately lightweight. If it works, we can later extend it.

## Why S$1,200 is reasonable

The price is for solving an operational problem, not for “AI.”

If the system prevents even a few missed follow-ups, it can quickly pay for itself by:

- Avoiding angry clients
- Protecting trust and referrals
- Reducing stress
- Saving admin time
- Making active projects easier to manage
- Helping staff know what needs attention

For a jewelry business, one lost or unhappy custom client can easily cost more than S$1,200 in lost repeat business, referrals, or reputation damage.

## If she reacts to the price

### If she says “That’s expensive”

“Totally fair. I don’t want to sell you a fancy AI thing you don’t need. The way I’d think about it is: if missed follow-ups are costing you angry clients, stress, or lost repeat business, then we test whether this prevents those slips. We can start with a small pilot on 5 active projects and only continue if it clearly helps.”

### If she says “Can’t I just use ChatGPT?”

“Yes, but ChatGPT by itself won’t know your active jobs, deadlines, client status, or follow-up cadence. The value here is not the chatbot — it’s designing the workflow around your real business so the right things get captured and surfaced at the right time.”

### If she says “I’m not techy”

“That’s exactly why I’d keep it lightweight. You shouldn’t need to learn a complicated tool. The assistant should work like an admin helper: paste notes, get structured follow-ups, check what’s overdue.”

### If she says “I’m too busy to set this up”

“That’s why I’d do the setup. I’d only need a little time upfront to understand your workflow, then we test with a few real jobs. The goal is to reduce your admin load, not add another system for the sake of it.”

## Meeting script

### 1. Open with empathy

“Last time you mentioned missed deadlines, missed messages, and angry clients. That sounds incredibly stressful because the jewelry work is already detailed and high-touch — and then on top of that you’re carrying all the project management in your head.”

### 2. Reframe the problem

“I don’t think the issue is that you need more software. I think you need a simple follow-up system that makes sure every client job has a clear next action, deadline, and reminder.”

### 3. Introduce the offer

“I’ve been setting up AI-assisted workflows, and I think there’s a practical version for your business: a client follow-up rescue system. It would help track active jobs, extract next steps from notes/messages, remind you what needs attention, and draft client updates.”

### 4. Give concrete examples

“For example, if a client asks about a repair deadline, or you promise to check with a supplier, we capture that as a follow-up item. Then each day you can ask: what clients need an update today? What’s overdue? Which jobs are at risk?”

### 5. Make it safe

“I wouldn’t start with a big complicated system. I’d suggest testing it with 5–10 active projects for two weeks. If it doesn’t reduce stress or missed follow-ups, then we stop.”

### 6. State the price confidently

“My minimum for setting this up properly is S$1,200. That includes understanding your workflow, setting up the tracker and assistant, testing it with real projects, training you, and making adjustments during the pilot.”

### 7. Pause and listen

Then stop talking. Let her react.
